# Neovim Config

## Setup

1. Clone this repo

```sh
	git clone https://github.com/bernardmuller/nvim-config.git ~/.config/nvim
```

2. Download the "Hack" Nerdfont from the [NerdFonts Repo](https://github.com/ryanoasis/nerd-fonts "NerdFonts Repo") and save the .ttf file in "~/.local/share/fonts/NerdFonts/"

3. Install a truecolor terminal like "iTerm2" with homebrew

```sh
  homebrew install iTerm 2
```

4. Open Neovim

```sh
  nvim
```

note: Navigate to "/lua/bernard/plugin-setup.lua" to see all the plugins I use.
